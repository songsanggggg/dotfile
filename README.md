# songsanggggg Arch Linux Dotfiles

这个仓库保存了 `songsanggggg` 在 `Arch Linux` 上使用的一套个人配置文件，主要面向基于 `swayfx` 的 Wayland 桌面环境。

当前仓库包含的主要内容：

- `swayfx` 配置
- `waybar` 配置
- `kitty` 配置
- `mako` 配置
- `fuzzel` 配置
- `swaylock` 配置
- 一键安装与部署脚本 `install.sh`

这些配置主要用于快速还原或同步日常使用环境，包括：

- 窗口管理与快捷键
- Waybar 状态栏
- 通知、锁屏、启动器
- 剪贴板、截图、音量、亮度等常用桌面功能

## 使用方式

如果你想直接安装或部署这些配置，可以运行：

```bash
./install.sh
```

详细安装说明见 [INSTALL.md](./INSTALL.md)，快捷键说明见 [HOTKEYS.md](./HOTKEYS.md)。
