# SwayFX Hotkeys

This list matches [.config/sway/config](./.config/sway/config:1). The searchable `Super+/` cheatsheet is generated from `## @keybind: Category | Description` comments in that file.

This layout now follows the general shortcut style from `end-4/dots-hyprland`, adapted for `sway`.

## Core

- `Super+/`: open searchable keybind quick reference
- `Super+Return`: open `kitty`
- `Super+Space`: open `fuzzel`
- `Super+e`: open `thunar`
- `Super+q`: close focused window
- `Super+Shift+c`: reload sway config
- `Super+Ctrl+b`: restart `waybar`
- `Super+Shift+e`: exit sway with confirmation
- `Super+l`: lock screen
- `Super+Shift+l`: suspend system
- `Super+Shift+w`: download and apply today's Bing wallpaper
- `Ctrl+Alt+Delete`: open `wlogout`

## Focus And Move

- `Super+Arrow`: move focus with arrow keys
- `Super+Shift+Arrow`: move focused window with arrow keys
- `Super+a`: focus parent container
- `Super+Ctrl+Space`: toggle focus between tiling and floating

## Workspaces

- `Ctrl+Super+Left/Right`: switch to previous/next workspace
- `Super+PageUp/PageDown`: switch to previous/next workspace
- `Super+1..0`: switch to workspace 1..10
- `Ctrl+Super+Shift+Left/Right`: move focused window to previous/next workspace
- `Super+Shift+PageUp/PageDown`: move focused window to previous/next workspace
- `Super+Shift+1..0`: move focused window to workspace 1..10

## Layout

- `Super+b`: horizontal split
- `Super+Ctrl+v`: vertical split
- `Super+Ctrl+s`: stacking layout
- `Super+w`: tabbed layout
- `Super+Ctrl+e`: toggle split layout
- `Super+d`: toggle fullscreen
- `Super+f`: toggle global fullscreen
- `Super+Alt+Space`: toggle floating
- `Super+p`: pin floating window to all workspaces
- `Super+Alt+s`: send window to scratchpad
- `Super+s`: show scratchpad window
- `Super+Shift+-`: send window to scratchpad
- `Super+-`: show next scratchpad window

## Resize Mode

- `Super+r`: enter resize mode
- `Arrow`: resize in resize mode
- `Return` or `Esc`: leave resize mode

## Clipboard, Screenshot, Notifications, Media

- `Print`: full screenshot with `grim`
- `Super+Shift+s`: area screenshot and open in `swappy`
- `Super+v`: open clipboard history through `fuzzel`
- `Super+Shift+v`: open `pavucontrol`
- `Super+Ctrl+n`: dismiss all `mako` notifications
- `Super+Shift+b`: previous track
- `Super+Shift+p`: play or pause
- `Super+Shift+n`: next track

## Audio And Brightness

- `XF86AudioMute`: mute output
- `XF86AudioLowerVolume`: volume -5%
- `XF86AudioRaiseVolume`: volume +5%
- `XF86AudioMicMute`: mute microphone
- `XF86AudioPlay`: play or pause media
- `XF86AudioPause`: pause media
- `XF86AudioNext`: next track
- `XF86AudioPrev`: previous track
- `XF86MonBrightnessDown`: brightness -5%
- `XF86MonBrightnessUp`: brightness +5%
